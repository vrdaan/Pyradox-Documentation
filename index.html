Pyradox: A Revolutionary Personal Digital Assistant

Pyradox is designed to get things done from basic assitance to assistance in data analysis and natural language processing and many other utilities.
The motivation behind Pyradox is to ease the tasks of a programmer by automating most of the frequently performed tasks

Discover what Pyradox can do

Virtual Assistant

Pyradox knows everything...
Test its knowledge by asking anything
Sounds exciting!!! Just give this a try..

Usage:
$ --question

Word Cloud

Visualising your text with some cool word clouds is pretty much exciting. Check it out now!

Usage:
$ wordcloud --w --h --mxw --bgc

Optional: provide an image mask for the cloud

Usage:
$ wordcloud --w --h --mxw --bgc --imgsrc

w: width

h: height

mxw: maximum number of words for the cloud

bgc: background color

imgsrc: mask image source

Data Analysis Features

Correlation

Plot a heatmap to visualize the correlation bwtween all the attribute

Usage
>> corr

Describe

Calculate various metrics such as mean, median, mode, min, max, sum, count, std, quantile, correlation with target column of a feature column

Usage
>> describe --column

Groupby

Generate a groupby table of columns c using aggregate function f

Usage
>> groupby --by --c --f

Drop, Outliers, Interquantile Range

Drop the outliers based on inter quantile range

Usage
>> drop --o --iqr

Drop, Outliers, Custom Quantiles

Drop the outliers which lie outside the mentioned quantiles

Usage
>> drop --o --q --beg --end

Drop, Outliers, Z-Score

Drop outliers in the data based on the zscore value (n)

Usage
>> drop --o --z --n

Drop, Null

Drop rows containing null values

Usage
>> drop --na

Insight

Calculate metrics such as mean, median, mode, min, max, sum, count, std, quantile, correlation with target column (corr) of the feature column

Usage
>> insight --col --metric

Drop, Columns

Drop a specific column

Usage
>> drop --col

Encode, Label

Label encode the attribute column

Usage
>> encode --l --col

Missing Data

Plot a heatmap to visualize the missing data

Usage
>> missingdata

Encode, One Hot

One hot encode the attribute column

Usage
>> encode --o --col

Done

Exit the data analysis mode and save the python for the analysis performed in the session and the modified dataset

Usage
>> done

Head

Shows the first n rows the dataset

Usage
>> head --n

Shows header of the dataset (first 0 rows)

Usage
>> head

Reload

Reload the original dataset

Usage
>> reload

Revert

Revert back to the last saved checkpoint

Usage
>> revert

Shape

Shows the shape of the dataset

Usage
>> shape

Save

Save the python for the analysis performed in the session and the modified dataset, thus create a checkpoint

Usage
>> save

Tail

Shows the last n rows the dataset

Usage
>> tail --n

Shows header of the dataset (last 0 rows)

Usage
>> tail

Unique

Shows the unique values of the attribute

Usage
>> unq --column

Value Distribution

Shows the value count of the attribute

Usage
>> valdist --column

Shows the value count of the attribute differentiated on th basis of various classes in the target column

Usage
>> valdisty --column

Pivot

Usage

   --i:index --c:columns --v:values

Plot, Bar

Plot an estimate of central tendency for a numeric variable with the height of each rectangle and provides some indication of the uncertainty around that estimate using error bars

Usage
>> plot --bar --x --y --estimator--hue(optional)

Impute

Impute the missing data in the attribute

Usage
>> impute --col --method

Methods:
¤ ffill
    Impute using forward filling
¤ bfill
    Impute using backward filling
¤ interpolate
    impute using interpolation
¤ mean --g:col
    impute the mean (optional: groupedby a particular column)
¤ median --g:col
    impute the median (optional: groupedby a particular column)
¤ mode --g:col
    impute the mode (optional: groupedby a particular column)

Impute the missing data in the attibute with a scalar (optional: numeric)

Usage
>> impute --col --value --n(optional)

Plot, Box

Draw a box plot to show distributions with respect to categories

Usage
>> plot --box --x --y --hue(optional)

Plot, Count

Show the counts of observations in each categorical bin using bars

Usage
>> plot --count --x --hue(optional)

Plot, Dist

Plot a univariate distribution of observations

Usage
>> plot --dist --col --bins(optional)

Plot, Joint

Draw a plot of two variables with bivariate and univariate graphs, kinds: scatter, reg, resid, kde, hex

Usage
>> plot --joint --x --x --kind(optional)

Plot, Pair

Plot pairwise relationships in a dataset, hue: variable in data to map plot aspects to different colors.

Usage
>> plot --pair --hue(optional)

Plot, Pie

Plot a circular statistical graphic which is divided into slices to illustrate numerical proportion

Usage
>> plot --pie --col

Plot, Violin

Draw a combination of boxplot and kernel density estimate

Usage
>> plot --violin --x --y --hue(optional)
