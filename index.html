Pyradox a revolutionary personal digital assistant

Pyradox is designed to get things done from basic assitance to assistance in data analysis and natural language processing and many other utilities.
The motivation behind Pyradox is to ease the tasks of a programmer by automating most of the frequently performed tasks

Discover what Pyradox can do

Syntax Search

Started learning a new language...
It's difficult to remember the syntax...
The syntax search is there for you
Search for a syntax on the syntaxdb website...

Usage:

$ syntax --query

Google Search

No need to go to chrome for searching anything
Search anything quickly on Google just by a single click of this feature.

Usage:

$ google --query

Word Cloud

Visualising your text with some cool word clouds is pretty much exciting. Check it out now!

Optional: provide an image mask for the cloud

Usage:

$ wordcloud --w --h --mxw --bgc
$ wordcloud --w --h --mxw --bgc --imgsrc

w: width

h: height

mxw: maximum number of words for the cloud

bgc: background color

imgsrc: mask image source
