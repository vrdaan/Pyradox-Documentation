Pyradox a revolutionary personal digital assistant

Pyradox is designed to get things done from basic assitance to assistance in data analysis and natural language processing and many other utilities.
The motivation behind Pyradox is to ease the tasks of a programmer by automating most of the frequently performed tasks

Discover what Pyradox can do

Syntax Search

Search for a syntax on the syntaxdb website...

Usage:

$ syntax --query

Google Search

Quick search anything on google...

Usage:

$ google --query

Calculate

Calculate some expression with the help of a google search...

Usage:

$ calc --expression

Browse

Quickly open a website just by its name...

Usage:

$ browse --website name

Me

Quickly browse your social media account like facebook, instagram, whatsapp, twitter, linkedin, github, codechef, codepen...

Usage:

$ me --social media site

Save webpage

Download a webpage to a pdf file...

Usage:

$ saveweb --url --path the pdf file

Walk

Perform a directory walk...

Usage:

$ walk --directory path

Memo

Organise your tasks efficiently

List all the saved memos...

Usage:

$ memo_all

Usage:

$ memo_new --memo title

Read the memo...

Usage:

$ memo_read --memo title

Delete the memo...

Usage:

$ memo_del --memo title

Delete all memos...

Usage:

$ memo_delall

Speech

Word Cloud

Visualize the text by creating some cool wordclouds...

Optional: provide an image mask for the cloud

Usage:

$ wordcloud --w --h --mxw --bgc
$ wordcloud --w --h --mxw --bgc --imgsrc

w: width

h: height

mxw: maximum number of words for the cloud

bgc: background color

imgsrc: mask image source
